Оптический баллистический хронограф измеряет скорость пуль и быстролетящих объектов с помощью двух оптических барьеров.
Устройство использует Arduino Nano, 4 ИК-светодиода и 28 фототранзисторов. Расстояние между датчиками - 10 см. 
Скорость вычисляется по формуле v = 0.1 / Δt, где Δt - время пролета между барьерами. 
Проект разработан студентами МФТИ для лабораторных работ и тестирования пневматического оружия.
